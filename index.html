Panduan Langkah-demi-Langkah

Apa itu ID Masjid?

ID Masjid adalah "Kode Unik" (seperti frekuensi radio) yang menghubungkan HP Admin DKM dengan ribuan HP Jamaah.

ID: MSJD-XXXX

*Artinya: Admin cukup input jadwal sekali, otomatis muncul di semua HP jamaah yang memasukkan ID tersebut.

Langkah 1: Masuk Sebagai Admin

Wajib

Sebelum mengubah data, Anda harus masuk ke sistem.

01
Buka Menu & Login
Klik tombol Menu (garis tiga) di pojok kiri atas -> Klik tombol "Login Admin" di paling bawah.
02
Masukkan Password
Password bawaan aplikasi adalah: admin123
*Segera ganti password ini di menu Pengaturan setelah login.

Langkah 2: Aktivasi Cloud (Server Online)

Fitur Unggulan

Agar jadwal & keuangan bisa muncul di HP Jamaah, Anda perlu mengaktifkan Cloud.

A. Buka Pengaturan

Masuk menu Pengaturan & Sync, scroll ke bawah cari kotak "Admin Cloud".

B. Aktivasi / Beli

Pilih paket (Bulanan/Tahunan) atau masukkan Kode Voucher jika punya.

C. Dapatkan ID Masjid

Setelah aktif, Anda akan dapat ID Masjid (Contoh: MSJD-1234). ID inilah kuncinya.

Langkah 3: Input Data Masjid

Setelah login dan aktif, saatnya mengisi konten.

Mengisi Jadwal
  • Menu Jadwal > Tab Harian (Untuk Imam Rawatib).
  • Isi form Imam Jumat jika hari Jumat.
  • Menu Jadwal > Tab Ramadhan (Untuk Tarawih/Kultum).
Mengisi Keuangan
  • Menu Keuangan.
  • Pilih Tanggal & Jenis (Masuk/Keluar).
  • Masukkan nominal dan keterangan, lalu klik .

Langkah 4: Tampilkan ke Jamaah

Data sudah siap? Tampilkan dengan 2 cara:

Cara 1: Ke HP Jamaah (Otomatis)

Umumkan ID Masjid Anda (dari Langkah 2) kepada jamaah. Jamaah cukup memasukkan ID tersebut di HP mereka sekali saja. Data akan sinkron selamanya.

Cara 2: Layar TV Masjid

Hubungkan Laptop ke TV Masjid via HDMI. Masuk menu Pengaturan -> Pilih Mode -> Klik Mulai TV Mode. Tampilan akan Full Screen dan berjalan otomatis.
